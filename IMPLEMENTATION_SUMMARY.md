## 🚀 如何运行

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动项目**
   ```bash
   npm start
   ```

3. **选择平台**
   - 按 `i` 启动iOS模拟器
   - 按 `a` 启动Android模拟器
   - 按 `w` 启动Web版本
